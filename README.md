<h2>Como usar</h2>

   ```
   Configure o ambiente de desenvolvimento na sua máquina:
   https://flutter.dev/docs/get-started/install

   - Clone o repositório:
   $ git clone https://github.com/oapmartins/vakinha_burger

   - Entre no diretório:
   $ cd vakinha_burger

   - Instale as dependências:
   $ flutter pub get

   - Execute:
   $ flutter run
   ```

   ```

   $ git checkout -b my-feature
   ```
